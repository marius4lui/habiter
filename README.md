# Habiter

This is the main Flutter application for Habiter.
It supports both Mobile (Android, iOS) and Web from a single codebase.

## Getting Started

Fetch dependencies:
```sh
flutter pub get
```

Run the app:
```sh
flutter run
```

## Structure
- `lib/`: Dart code (screens, models, providers, etc.)
- `web/`: Web entrypoint
- `android/`: Android native code
- `ios/`: iOS native code
