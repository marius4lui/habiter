# Features

## Habit Tracking

Users can create, edit, and delete habits. Each habit can have:

-   **Name**: The title of the habit.
-   **Frequency**: How often it should be performed (Daily, Weekly, etc.).
-   **Goal**: Target number of completions.

## Notifications

The notification system ensures users don't forget their habits.

-   **Reminders**: Specific time reminders for each habit.
-   **Quick Actions**: Interact with notifications directly.

## Theming

Habiter supports multiple themes, primarily focusing on a "Cozy Dark Mode" to reduce eye strain.
