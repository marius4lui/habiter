# Getting Started

This guide will help you set up the Habiter project on your local machine.

## Prerequisites

- **Flutter SDK**: Ensure you have the latest stable version of Flutter installed.
- **Dart SDK**: Included with Flutter.
- **IDE**: VS Code or Android Studio with Flutter extensions.

## Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/marius4lui/habiter.git
    cd habiter
    ```

2.  **Install dependencies:**

    ```bash
    flutter pub get
    ```

3.  **Run the app:**

    ```bash
    flutter run
    ```

## Running Tests

To run the unit and widget tests:

```bash
flutter test
```
